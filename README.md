## Connecd

A social media app built with React.

## Get started

### Clone the repo

```
 git clone https://github.com/HanMoeHtet/connectd.git
```

### Install dependencies

```
 npm install
```

### Copy env file and fill in the values

```
  cp .env.example .env
```

###  Run the app

```
 npm run dev
```
